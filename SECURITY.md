# Security Policy

## Supported Versions

Only 5.0 release is currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 5.0.x   | :white_check_mark: |
| < 5.0   | :x:                |

## Reporting a Vulnerability

Please contact [LeXtudio Inc.](https://lextudio.com) directly to report a vulnerability.
